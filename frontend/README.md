# Frontend - Todo App

This is the frontend for the Todo App built using React, TypeScript, and TanStack Query for managing server-side state. It connects to a backend that handles CRUD operations on a SQLite database.

## Features

- Fetch Todos from the backend
- Add new Todos
- Update existing Todos (mark as completed)
- Delete Todos
- State management using TanStack Query

## Prerequisites

- Node.js (>= 12.x)
- Backend server (see [Backend README](../backend/README.md))

## Setup

1. **Install dependencies**:

   ```bash
   npm install
   ```


2. **Start the development server**:

   ```bash
   npm run dev
   ```

3. **Access the app**:

   ```bash
   The app will be running on http://localhost:5173/
   ```
